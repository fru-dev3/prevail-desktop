# Quarterly skills audit

Run once a quarter, or before planning any course or certification.

1. **Moved.** From _state.md and recent threads: which skills actually got
   exercised this quarter? Shipped beats studied.
2. **Market.** Compare against what current postings for the next role demand.
   Name gaps honestly, including ones that are unfashionable to admit.
3. **Prune.** Identify one thing to STOP investing in: a skill the market has
   stopped paying for, or one already at diminishing returns.
4. **One bet.** Pick the single skill investment for next quarter with the best
   payoff for the target role, and define what done looks like in 90 days.

Output: moved / gaps / prune / the one bet with its 90-day definition of done.
