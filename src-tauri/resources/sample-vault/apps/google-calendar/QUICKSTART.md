# Google Calendar — Quickstart

Use this app whenever you need calendar truth instead of an old vault snapshot.

## Common openers

- *"What's on my calendar this week?"* — pulls primary + work calendars and
  summarizes meeting load, deep-work blocks, conflicts
- *"Next free 90-minute block?"* — scans the next 7 days for an unscheduled
  window long enough for focused work
- *"Conflict-check this invite: <details>"* — checks against existing events
  and the no-meetings rules
- *"Block Friday Apr 17 as a no-meetings day"* — creates an all-day busy
  event on all calendars
