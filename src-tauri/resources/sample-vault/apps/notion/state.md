# Notion — Alex Rivera (Demo)

> **SYNTHETIC DATA — Demo only.**

**Used by domains:** content, brand, career, vision
**Workspaces:** 2 (Personal, Stripe-Interview-Prep)
**Auth:** integration token (no expiry)

## What Lives Where

| Database | Workspace | Used for |
|----------|-----------|----------|
| Content Calendar | Personal | brand + content planning, YT pipeline |
| Newsletter Drafts | Personal | unpublished newsletter posts |
| Career Pipeline | Personal | interview prep, target roles, contacts |
| System Design Notes | Stripe-Interview-Prep | scratch space for SD review |
| Life Vision | Personal | 1y / 3y / 10y goal trackers |

## When to Use This App

- Append a draft to Content Calendar without leaving the cockpit
- Pull career pipeline status into the Chief brief
- Sync vault/vision/state.md to the Life Vision DB monthly
