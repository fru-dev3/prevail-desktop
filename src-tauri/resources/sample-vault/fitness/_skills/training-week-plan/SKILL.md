# Training week plan

Run Sunday, against data/training-log.csv and the half-marathon plan thread.

1. **What actually happened.** Planned vs completed sessions this week. No
   judgment, just the delta and the reason.
2. **Recovery check.** Any soreness, poor sleep, or life stress that argues for
   a lighter week? Adaptation happens in recovery, not in the workout.
3. **Next week.** Set the sessions: one long, one quality (tempo or intervals),
   the rest easy. Total load no more than 10% above this week.
4. **The skip rule.** Decide in advance which session gets dropped if the week
   goes sideways, so the long run is never the casualty.

Output: the week's sessions with distances/paces and the designated skip.
