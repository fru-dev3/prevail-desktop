## Prompt

I want to run a half marathon in about 16 weeks. Is that realistic from my current base, and how should I train?

## Context

Currently running ~10 miles/week across 3 sessions, longest recent run 5 miles. Busy work schedule, history of starting ambitious plans that collapse.

## Notes

A good answer says 16 weeks is realistic from this base with a conservative build (one long run, one quality session, easy rest, ~10% weekly load cap) and a designated skip session so the long run is never the casualty.
