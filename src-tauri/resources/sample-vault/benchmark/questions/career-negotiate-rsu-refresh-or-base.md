## Prompt

It's been 18 months since my last compensation review. Should I push for a base salary increase or an RSU refresh grant?

## Context

Current TC is competitive for the role. RSUs from original grant are 50% vested. Company is doing well. Manager relationship is strong. Stock price is up ~20% from grant date.

## Notes

A good answer should recommend leading with an RSU refresh (recurring comp, tax-advantaged upside, ties to company performance) and using base as a fallback ask. Should explain the tax timing and vesting cliff considerations.
