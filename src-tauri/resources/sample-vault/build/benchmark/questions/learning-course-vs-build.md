## Prompt

I keep buying courses to skill up for the Staff promo but never finish them. What should I actually do?

## Context

Alex has several unfinished courses. Wants to demonstrate Staff-level system design. Limited evening time.

## Notes

A good answer reframes toward a shippable artifact (a real system-design project or write-up) over passive course consumption, scoping the learning to serve that artifact and scheduling a few protected sessions a week.
