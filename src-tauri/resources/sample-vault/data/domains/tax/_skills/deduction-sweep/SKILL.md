# Deduction sweep

Run in November, while there is still time to act.

1. **Standard vs itemized.** Run the totals honestly. If itemizing loses by a small margin, check whether bunching (two years of charity in one) flips it.
2. **Above-the-line.** Retirement contributions, HSA, FSA: which still have room this year, and the deadline for each.
3. **Credits.** Walk the credit list against this year's facts (energy improvements from the homestead domain, education from learning).
4. **Evidence.** For everything claimed: the document that proves it, filed in the tax folder now, not in April.

Output: the standard-vs-itemized verdict, the room-left list with deadlines, and the evidence checklist.
