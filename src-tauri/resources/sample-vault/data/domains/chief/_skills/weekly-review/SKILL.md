# Weekly review

Run Friday across ALL domains, closing the week before it bleeds into the next.

1. **What moved.** Sweep each domain's _state.md and recent threads for what actually progressed this week. Name real movement, not activity.
2. **What stalled.** Flag the commitments that didn't move and why — blocked, dropped, or quietly abandoned. Honesty here is the whole point.
3. **Loops opened and closed.** Reconcile each domain's open-loops.md: what closed, what's newly open, what's been open too long and needs the cross-domain loop sweep next.
4. **One reset.** Pick the single adjustment for next week — a habit, a boundary, a priority — that would matter most. One, so it sticks.

Output: moved / stalled / loop status by domain, and the one reset for next week.
