# Network touchpoint plan

Run monthly, before the relationships go quiet.

1. **Going cold.** From data/contacts.csv and recent threads, flag the people worth keeping who haven't heard from you in 90+ days. Relationships decay silently.
2. **Prioritize.** Pick the handful that matter most this month — mentors, past collaborators, people one step ahead on the path you want. Quality of tie over headcount.
3. **Give before ask.** For each, find the genuine reason to reach out: an intro, a useful link, congratulations on their news. No naked asks.
4. **Draft the openers.** Write the actual first message for each — short, specific, no "just checking in". Schedule the sends so it happens.

Output: the cold list, this month's chosen few, and a ready-to-send opener for each.
