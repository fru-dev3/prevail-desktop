# Weekly wins log

Run Friday afternoon, while the week is still in memory.

1. **Harvest.** From _state.md, recent threads, and this week's shipped work, list what actually moved: things shipped, decisions unblocked, people helped. Skip the busywork.
2. **Quantify.** For each win, add the number that makes it real — time saved, scope grown, revenue or risk touched. "Helped the launch" becomes "cut deploy time 40%".
3. **Tag the level.** Mark which wins demonstrate the next role's scope, not just the current one. Those are the promotion fuel.
4. **Append, don't overwrite.** Add the entries to data/brag-doc.md under this week's date so the record compounds over the year.

Output: this week's dated wins, each with its quantified impact, appended to the brag doc.
