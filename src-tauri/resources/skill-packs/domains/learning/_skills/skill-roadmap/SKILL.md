# Skill roadmap

Run when a skill matters enough to deserve a path, not just a pile of courses.

1. **Name the gap.** Where you are now and the specific depth the next role or goal demands. Vague targets ("get better at X") never resolve; write the level you'd recognize when you reach it.
2. **Stage the climb.** Break the gap into 3–5 ordered milestones, each with a small proof of competence — a built thing, a passed test, a shipped piece of work.
3. **Attach the material.** Map each milestone to a course or book from data/courses.csv and data/books.csv; note where there's nothing yet and the gap to fill.
4. **Set the cadence.** A rough date per milestone and the first sprint to start now (hand off to the learning-sprint-plan skill).

Output: the staged roadmap with a proof and a target date per milestone, saved to data/roadmaps/.
