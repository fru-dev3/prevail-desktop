# Code of Conduct

This project follows the [Contributor Covenant](https://www.contributor-covenant.org/version/2/1/code_of_conduct/) v2.1.

In short: be respectful, assume good faith, and keep discussion constructive.
Harassment, discrimination, or hostility are not tolerated. Report concerns to
**conduct@example.com**; maintainers will respond and may remove contributors who
violate these norms.
